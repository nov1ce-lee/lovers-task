# lovers task - 情侣任务奖励系统 ❤️

这是一个专为情侣设计的任务奖励互动系统。通过完成任务赚取积分，兑换对方提供的甜蜜奖励，增进感情互动。

## ✨ 特色功能

*   **双人角色**：内置 "男生" (Boy) 和 "女生" (Girl) 两个角色，通过点击按钮直接登录，无需密码。
*   **任务系统**：
    *   发布任务：双方都可以给对方发布任务（支持单次任务和每日打卡）。
    *   任务打卡：完成任务后提交打卡，等待对方审核。
    *   审核机制：发布方审核通过后，执行方获得金币奖励。
*   **奖励商城**：
    *   上架奖励：可以将“陪我看电影”、“按摩券”等作为商品上架。
    *   积分兑换：使用赚取的金币兑换奖励。
*   **极简交互**：专为移动端优化的界面，操作简单直观。
*   **一键切换**：方便的身份切换功能，便于测试或共用设备。

## 🛠️ 技术栈

*   **后端**：Python + FastAPI
*   **数据库**：SQLite + SQLModel
*   **前端**：Vue 3 + Tailwind CSS (单页应用)
*   **部署**：Docker / 本地运行

## 🚀 快速开始

### Windows 本地运行

1.  **克隆项目**
    ```bash
    git clone <your-repo-url>
    cd TaskReward
    ```

2.  **安装依赖**
    ```bash
    pip install -r requirements.txt
    ```

3.  **启动服务**
    双击运行 `run_windows.bat` 脚本，或者在终端运行：
    ```bash
    python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
    ```

4.  **访问应用**
    打开浏览器访问：[http://localhost:8000](http://localhost:8000)

### Docker 部署

```bash
docker-compose up -d --build
```

## 📝 使用指南

1.  **初次进入**：在首页选择 "👦 我是男生" 或 "👧 我是女生"。
2.  **发布任务**：点击 "发布任务"，填写标题、描述和奖励金额。
3.  **上架奖励**：点击 "上架商品"，填写奖励内容和所需积分。
4.  **互动**：完成对方布置的任务，或者用积分为对方兑换奖励！

## 🎨 设计理念

拒绝繁琐的注册登录流程，专注于情侣间的甜蜜互动。界面风格温馨，操作流畅。
